# LX Compass++

LX Compass++ is an addon that allows you to locate structures and biomes anywhere in the world. If you're looking for a compass that can locate biomes, this addon does both!

## Features

- **Right-click** with the LX Compass to open a GUI where you can select a structure or biome to search for
- Supports **all vanilla structures** across all three dimensions: Overworld, Nether, and The End
- Supports **all vanilla biomes** in every dimension
- **Search function** to quickly find specific structures or biomes by name
- **Multi-language support**: English, Spanish, Portuguese, Japanese, and Russian
- **Automatic delivery**: Players receive the compass automatically on first spawn
- **Keeps on death**: The compass is never lost
- Works in any world - no beta APIs required!

## How to Use

1. **Get the Compass**: The LX Compass is automatically given to players when they first spawn in a world
2. **Open the Menu**: Right-click with the compass to open the selection GUI
3. **Select Dimension**: Choose between Overworld, Nether, or The End
4. **Choose Category**: Select whether you want to search for Structures or Biomes
5. **Find Target**: Browse the list or use the search function to find what you're looking for
6. **Follow the Direction**: The game will show you the direction and distance to your target!

## Recipe

The crafting recipe for the LX Compass:

```
| Gold Ingot | Empty Map | Gold Ingot |
| Empty Map  | Compass   | Empty Map  |
| Gold Ingot | Empty Map | Gold Ingot |
```

Required materials:
- 1 Compass
- 5 Empty Maps
- 4 Gold Ingots

## Installation

1. Download the addon
2. Open Minecraft Bedrock Edition
3. Go to Settings > Storage
4. Import the .mcpack file
5. Create a new world or add to existing world
6. Enable "Player Analytics" in world settings (optional but recommended)
7. Play!
